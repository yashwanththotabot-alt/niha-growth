## ðŸ“– What is HTML?

HTML = **HyperText Markup Language**

- It's the skeleton of every website
- Defines structure: headings, paragraphs, links, images
- Browser reads HTML and displays it as a webpage

---

## ðŸ§  Key Concepts

### 1. HTML Elements
```html
<h1>This is a heading</h1>
<p>This is a paragraph</p>
<a href="https://google.com">This is a link</a>
<img src="image.jpg" alt="Description">
```

### 2. Document Structure
```html
<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>Hello World!</h1>
        <p>This is my first page.</p>
    </body>
</html>
```

### 3. Common Tags

| Tag | Purpose | Example |
|-----|---------|---------|
| `<h1>` to `<h6>` | Headings | `<h1>Big</h1>` |
| `<p>` | Paragraph | `<p>Text here</p>` |
| `<a>` | Link | `<a href="url">Text</a>` |
| `<img>` | Image | `<img src="pic.jpg">` |
| `<ul>`, `<ol>`, `<li>` | Lists | `<ul><li>Item</li></ul>` |
| `<div>` | Container | `<div>Group</div>` |
| `<span>` | Inline text | `<span>text</span>` |
